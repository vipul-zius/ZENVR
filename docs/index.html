Projective Module Theorem Explorer

Interactive exploration of projective module theorems in commutative algebra

Theorem 1

Lifting Property

P is projective iff given any surjection ψ: M → N and map ν: P → N, ∃ μ: P → M such that ψ∘μ = ν

Proof: The functor Hom(P,–) is exact iff it preserves surjections.
Theorem 2

Splitting Property

P is projective iff every short exact sequence 0 → L → M → P → 0 splits

Proof: Apply lifting property to id: P → P to get splitting map.
Theorem 3

Direct Summand of Free

P is projective iff P is a direct summand of a free module F

Proof: Take free module F mapping onto P, sequence splits by Theorem 2.
Theorem 4

Local Ring Theorem

Over a local ring A, a finitely generated module P is projective iff it is free

Proof: Use Nakayama's lemma and the fact that projective modules over local rings are free.
