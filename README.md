# Quantum Flight Search Application
Quantum algorithm application to search for optimal flights between Dublin and India.

## Features
- 24-qubit quantum optimization for flight routing
- Machine learning for passenger profile analysis
- Real-time flight data integration
- Advanced visualization dashboard

## Tech Stack
- Quantum: Qiskit, Cirq (24-qubit simulation)
- ML: scikit-learn, pandas, numpy
- Backend: Python 3.10, FastAPI
- Frontend: React/Next.js (planned)
