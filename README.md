# M-DOD: Multi-Domain Operations Dashboard

## Project Overview
A secure, unified software platform for tactical decision support that fuses real-time data across five domains (land, sea, air, space, cyber) with AI-powered analytics.

## Quick Start
```bash
# Clone and setup
git clone <repository-url>
cd mdod-project

# Run system check
./system_check/mdod_system_monitor.sh

# Install dependencies
./scripts/install_crucial_tools.sh

# Deploy
./scripts/deploy_orchestrate.sh
Architecture
Frontend: React/Vite TypeScript dashboard

Backend: FastAPI Python microservices

Data Processing: Go stream engine

Security: JWT-based API gateway

Monitoring: Prometheus/Grafana stack
